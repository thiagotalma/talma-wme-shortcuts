Acertar Rua

O que ele faz?
Simula você clicando e alterando o nome das ruas.
Corrige abreviações de acordo com a tabela do wiki e acentua palavras comuns, como nomes de pessoas.
Exemplo:
Nome antes: Rua Professor Antonio De Almeida E Silva
Corrigido: R. Prof. Antônio de Almeida e Silva

Como usar?

  1. Criar um bookmarklet (?). A forma mais fácil é arrastar o botão "Acertar Rua" acima até sua barra de favoritos;
  2. Vá no WME. Se for a primeira vez, force o recarregamento da página apertando ctrl + F5;
  3. Clique no favorito e aguarde ele carregar o script. Um alerta é mostrado quando carregar com sucesso;
  4. Selecione um segmento que precisa ajustar a abreviação;
  5. Aperte ctrl + Q;
  6. Observe a alteração e confira se está tudo certo;
  7. Aperte o botão para salvar.

Como ele trabalha?

  1. "Magicamente" clica para selecionar toda a via;
  2. Clica para editar os segmentos;
  3. Faz as substituições necessárias no nome da via;
  4. Clica para aplicar.

O que ele não faz?

  • Salvar. Nada é salvo automaticamente para que você possa conferir se está tudo certo e então manualmente clicar para salvar;
  • Alteração em massa. O script é uma AJUDA para editar, não é um automatizador do WME.

Se acharem que dá para ajustar mais coisas é só sugerirem que incluo no script.
